Campaign Overview

A mist-shrouded world where players traverse shifting interconnected locations between sessions. Sacred Flames keep the mist at bay while the party explores The Path - a dynamic network of locations that changes with each journey.

Quick Reference

Core Mechanics

  • Sacred Flame Resource: Village consumes 1 Flame Essence per day
  • Path Generation: Tarot-based card drawing system
  • Location Stability: Stable (persistent) or Unstable (disappear between sessions)
  • Underground Rules: Underground locations connect only to other Underground or Cave Mouth locations
