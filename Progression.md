3. **Framework Foundation**: Building core systems before content
4. **Integration Focus**: Ensuring compatibility with existing agents/SRD
5. **Binary Stability**: Simplified location states to stable/unstable only
6. **Player Agency**: Tarot card system gives players control over path creation
7. **Resource Management**: Sacred Flame system adds strategic maintenance layer
