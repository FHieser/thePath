# The Path - Campaign Overview

## Core Campaign Concept

**"The Path"** is a Daggerheart campaign frame built around exploration, mystery, and evolving locations in a mist-shrouded world where normal travel is impossible.

---

## **The World**

### **The Mist**
An otherworldly phenomenon has engulfed the fantasy world, making traditional overland travel deadly. The Mist:
- Disorients and confuses travelers who enter unprepared
- Hides dangerous creatures and unstable magic
- Changes the landscape itself, making maps useless
- Can only be safely navigated via specific routes

### **The Path Network**
A mysterious system of interconnected locations linked by walkways that shift between sessions:
- **Stable Connections**: Some links persist across sessions
- **Shifting Routes**: Other connections change based on unknown forces
- **Sacred Flames**: Ancient fires mark safe havens and push back the Mist
- **Memory**: Locations remember visits and evolve based on player actions

---

## **Campaign Structure**

### **Session Flow**
1. **Village Phase**: Players begin each session in Village in the Mist
   - Resupply, rest, gather information
   - Interact with NPCs and pursue storylines
   - Choose their next Path destination

2. **Path Phase**: Journey through interconnected locations
   - Navigate shifting routes and environmental challenges
   - Encounter adversaries, mysteries, and opportunities
   - Make decisions that will affect future visits
   - **TODO**: Determine if players choose their own path or GM selects

3. **Return Phase**: Journey back to Village (or another safe haven)
   - Process consequences of Path exploration
   - Level up, distribute rewards, advance storylines
   - Path reconfiguration occurs between sessions

### **Path Reconfiguration**
Between sessions, the GM:
- Updates location states based on player actions
- Shifts some Path connections while keeping others stable
- Introduces new locations and removes temporarily inaccessible ones
- Advances NPC storylines and world events

---

## **Core Mechanics**

### **Location Memory System**
Each location tracks:
- **Visit History**: How many times players have been there
- **Player Actions**: Significant choices and their consequences
- **State Changes**: How the location has evolved
- **NPC Relationships**: How residents remember the party

### **Mist Interaction Rules**
- **Safe Zones**: Areas protected by Sacred Flames
- **Traversal**: Following established Path connections is safe
- **Deviation**: Leaving marked routes risks Mist encounters
- **Mist Marks**: Accumulated exposure has mechanical effects

## **Campaign Themes**

### **Exploration & Discovery**
- Unknown territories with shifting geography
- Ancient mysteries hidden in the Mist
- Rewards for bold exploration balanced with real risks

### **Consequence & Memory**
- Player choices have lasting impact on locations
- NPCs and environments remember past interactions
- Building relationships and reputation over time

### **Community & Sanctuary**
- Village in the Mist as safe haven and social hub
- Building connections with fellow travelers
- Protecting sanctuaries from Mist encroachment

### **Mystery & Wonder**
- Origins and purpose of the Path network
- Nature and source of the otherworldly Mist
- Ancient civilizations and their abandoned sites

## **Scaling Guidelines**

### **Tier 1 (Level 1): Path Walkers**
- **Focus**: Learning Path navigation and basic survival
- **Scope**: Single Path routes with 3-5 connected locations
- **Mysteries**: Local legends and recent disappearances
- **Threats**: Mist creatures, unstable terrain, resource scarcity

### **Tier 2 (Levels 2-4): Path Shapers**
- **Focus**: Influencing Path configurations and local politics
- **Scope**: Multiple Path networks with branch routes
- **Mysteries**: Ancient ruins and pre-Mist civilization
- **Threats**: Organized adversaries, territorial conflicts, Mist storms

### **Tier 3 (Levels 5-7): Path Guardians**
- **Focus**: Protecting Path network and major sanctuaries
- **Scope**: Region-wide Path systems with multiple Villages
- **Mysteries**: Origin of the Mist and Sacred Flames
- **Threats**: Mist corruption, faction wars, reality distortions

## **Integration with Daggerheart**

### **Core System Compatibility**
- Uses standard Daggerheart rules without modification
- Leverages existing environment, adversary, and NPC structures
- Incorporates Hope/Fear mechanics into Path navigation
- Utilizes community creation rules for Village development

### **Homebrew Elements**
- **Path Network Mechanics**: Custom rules for location connections
- **Mist Interaction System**: Environmental hazard mechanics
- **Location Memory Rules**: Tracking and applying visit consequences
- **Village Hub Systems**: Extended downtime and social mechanics

## **Getting Started**

### **GM Preparation**
1. Read this overview and understand core concepts
2. Review Village in the Mist baseline content
3. Choose or generate first Path configuration
4. Prepare 2-3 locations for initial exploration
5. Create 3-5 Village NPCs with basic motivations

### **Player Introduction**
1. Begin in Village in the Mist during character creation
2. Establish how characters arrived at the sanctuary
3. Introduce Path concept through NPC interactions
4. Plan first Path expedition as group decision
5. Set expectations for evolving world and consequences

### **First Session Goals**
- Establish Village as safe home base
- Successfully navigate first Path route
- Experience location memory and NPC relationships
- Make meaningful choices with future consequences
- Return safely with sense of accomplishment and mystery
