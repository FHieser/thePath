# Upgrades

These are the upgrades that can be applied to the village in the mist.
They are defined by:
- Name: A short name for the upgrade
- Improvement: The improvement that the upgrade provides
- Provider: The person or organization that provides the upgrade
- Time Period: The time it takes to apply the upgrade
- Required Materials: The materials needed to apply the upgrade
- Notes: Any additional notes about the upgrade

## Upgrade List


- **Building Flame Canister:**:
  - Improvement: The village can construct a flame canister.
  - Provider: Hearth Keeper
  - Time Period: 1 day
  - Required Materials: Mithril Ore
  
- **Increase Sacred Flame Storage:**:
  - Time Period: 2 days
  - Required Materials: Make a traversable path to the stone-works location.
  - Improvement: The village's flame storage capacity is increased by 2.

- **Better Path Reconnaissance:**:
  - Improvement: The Path Reader can now read the village's path more effectively. Improve the Die Size for the path reconnaissance roll by 1 Size.
  - Provider: Path Reader
  - Time Period: 3 days
  - Required Materials: A item linked deeply to the mist.

- **Underground Access:**:
  - Improvement: Villagers can now access the underground network directly.
  - Provider: Hearth Keeper
  - Time Period: 3 days
  - Required Materials: None
  - If the players find a creature or person that can help in this endevour, decrease the time period to 1 day.

## Requirements Ideas:

- Treasure Hunt: find a clue in a location and follow it to a special location
