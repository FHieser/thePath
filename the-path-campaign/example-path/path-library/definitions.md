# Path Library

A collection of reusable path configurations for The Path campaign frame. Each file contains only the location grid — encounters, NPCs, and events are left to the GM to populate per session.

## Complexity Tiers

Tiers are based on total location count and overall path difficulty. Underground locations count equally toward the total. Underground paths can appear at any tier.

| Tier | Name | Description | Location Count |
|------|------|-------------|----------------|
| **1** | Scout | A short, focused route. Good for new players or a single-session sprint. | ~5–8 |
| **2** | Journey | A moderate-length path with some variety and choice. | ~8–11 |
| **3** | Expedition | A full, well-rounded path. Meaningful decisions, multiple zones. | ~11–14 |
| **4** | Campaign | A large, demanding path requiring careful resource management. | ~14–17 |
| **5** | Odyssey | Maximum scope. Sprawling, interconnected, and punishing. | ~17+ |
