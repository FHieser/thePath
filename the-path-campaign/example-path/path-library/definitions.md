# Path Library

A collection of reusable path configurations for The Path campaign frame. Each file contains only the location grid — encounters, NPCs, and events are left to the GM to populate per session.

## Rating System

Each path is rated on two axes:

**Daggerheart Tier (1–4):** Which campaign phase the path is designed for, matching the official Daggerheart tier progression.

| Tier | Name | Campaign Phase |
|------|------|----------------|
| **1** | Grassroots | Level 1. New characters, lower threat density. |
| **2** | Adventurer | Levels 2–4. Standard campaign range. |
| **3** | Champion | Levels 5–7. High threat, complex encounters. |
| **4** | Master | Levels 8–10. Maximum power, brutal paths. |

**Personal Ranking (1–10):** Real-play difficulty assessment — how punishing the path felt at the table. A `?` means the path hasn't been run yet.

File names use the format `[dh-tier].[personal-ranking]-[name].md` (e.g., `2.4-hollow-cross.md`).
