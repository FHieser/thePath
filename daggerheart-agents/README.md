# Daggerheart Agents

A comprehensive set of specialized agents for managing all aspects of the Daggerheart RPG system, built from the official System Reference Document (SRD).

## Agent Overview

### 1. Character Creator & Progression Agent
**Focus:** Character creation and advancement through all 10 levels
- Complete 9-step character creation process
- All 18 ancestries and 9 communities
- All 9 classes with subclasses and domain cards
- Leveling, multiclassing, and advancement optimization
- Equipment tier progression and build synergies

### 2. Combat Mechanics Agent  
**Focus:** Daggerheart's narrative combat system
- Duality Dice system (Hope/Fear mechanics)
- Action roll resolution and consequence management
- Damage thresholds and Hit Point tracking
- Conditions, range, movement, and tactical guidance
- Narrative turn order and spotlight management

### 3. Equipment Agent
**Focus:** Weapons, armor, loot, and economic systems
- Complete weapon and armor tables across all tiers
- Magic items and consumables management
- Gold system (handfuls/bags/chests)
- Crafting recipes and equipment progression
- Build optimization and upgrade timing

### 4. Encounters & Environments Agent (Merged)
**Focus:** Complete encounter design and management
- 300+ adversary stat blocks across 4 tiers
- 10 adversary types (Bruiser, Horde, Leader, Minion, etc.)
- Battle Point encounter balancing system
- Environmental challenges and dynamic battlegrounds
- Adversary-environment synergy and tactical coordination

### 5. Campaign Frames Agent
**Focus:** Long-term campaign structure and themes
- Campaign framework template system
- The Witherwild complete campaign example
- Theme integration and narrative structure
- Player/GM principles and session zero guidance
- Custom campaign creation tools

### 6. Adventure Running Agent
**Focus:** Complete GM support and session management
- GM Principles and collaborative storytelling
- Fear economy management and GM moves
- Scene pacing and tension escalation
- Player engagement and trust building
- Session preparation and improvisation support

## Context Sources

Each agent is built from extensive analysis of the Daggerheart SRD:
- **Core Rules:** Character creation, combat, equipment, leveling
- **Content Database:** 300+ adversaries, environments, weapons, armor, items
- **GM Guidance:** Principles, practices, mechanics, and session management
- **Campaign Material:** The Witherwild campaign frame and framework templates

## Key Features

- **Complete Coverage:** Every major system and mechanic in Daggerheart
- **Deep Integration:** Agents work together across all game aspects  
- **Optimization Focus:** Build advice and synergy recommendations
- **Narrative Support:** Emphasis on collaborative storytelling
- **Practical Tools:** Ready-to-use guidance for players and GMs

## Usage

These agents can be implemented individually or as a complete system to provide comprehensive support for Daggerheart campaigns. Each agent includes:

- Detailed capability descriptions
- Context source references
- Agent workflow processes  
- Key mechanics handled
- Special features and advanced techniques

Perfect for both new players learning the system and experienced groups optimizing their gameplay experience.
